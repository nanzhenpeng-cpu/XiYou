西柚数据清洗+ST打标工具

可直接在浏览器中运行

✨ 主要功能
📁 多文件上传 – 支持同时上传多个西柚关键词Excel文件、ASIN映射表、关键词数据表及自定义打标规则。
🧹 数据清洗 – 自动修正列名、去重搜索量、合并ASIN信息（品牌/机型/系列）。
🏷️ 智能打标 – 根据规则对关键词进行多标签打标（品牌词、竞品词、机型、功能词等），支持6种匹配模式（包含任一/全部、精确等于、前后缀、正则）。
📊 结果预览 – 实时预览清洗后的西柚数据、打标结果及关键词标签。
📤 一键导出 – 将清洗数据、打标结果及关键词打标分sheet导出为Excel。
📋 规则模板 – 内置可下载的标准打标模板，含13条常用规则及说明。

🚀 如何使用
1. 按需上传所需文件：
   - 西柚数据文件（可选，用于清洗打标）
   - ASIN Mapping（可选，关联产品信息）
   - 关键词数据表（可选，单独对关键词打标）
   - 打标规则表（必需，可先下载模板填写）
3. 点击 “开始处理”，等待数据处理完成。
4. 在标签页切换查看清洗结果、打标数据及关键词标签。
5. 点击 “导出结果” 下载最终Excel文件。
